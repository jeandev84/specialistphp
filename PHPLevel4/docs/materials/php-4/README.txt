Для работы на курсе требуется текстовый редактор.
Вы можете пользоваться своим, если таковой имеется.
Но в аудитории все демонстрации и лабораторные работы выполняются в редакторе Adobe Brackets.
Необходимо скачать версию специально настроенную под данный курс!!!
Скачать её можно по адресу: http://igor-borisov.ru/courses/link/brackets/
Программу не надо инсталлировать, просто распакуйте архив и запустите файл BracketsLauncher.exe

В исключительных ситуациях может использоваться текстовый редактор Notepad++
Скачать его можно по адресу: http://notepad-plus-plus.org/download/
