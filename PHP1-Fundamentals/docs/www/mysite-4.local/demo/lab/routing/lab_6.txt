Передача параметров на сервер

Упражнение 1: Создание подключаемых файлов

1. В текстовом редакторе откройте файл "about.php"
2. Удалите всё, кроме блока <!-- область основного контента -->
3. Сохраните файл about.php
4. В текстовом редакторе откройте файл "contact.php"
5. Удалите всё, кроме содержимого блока <!-- Область основного контента -->
6. Сохраните файл "contact.php"
7. В текстовом редакторе откройте файл "calc.php"
8. Удалите всё, кроме содержимого блока <!-- Область основного контента -->
8. Сохраните файл calc.php

Упражнение 2: Изменение меню

1. В текстовом редакторе откройте файл "data.inc.php"
2. Необходимо изменить значения во всех элементах "href" массив $lefMenu на "index.php"
   и добавить параметра (например "id"), который будет передавать методом "GET"
   уникальные значения

3. Измените значения, которые в результате могут выглядеть так:
   index.php
   index.php?id=about
   index.php?id=contact
   index.php?id=table
   index.php?id=calc

4. Сохраните файл "data.inc.php"

Упражнение 3. Приёмы данных от пользователя

1. В текстовом редакторе откройте файл "index.php"
2. В верхней части файла перед закрывающим тэгом ?> напишиите
// Инициализация заголовок страницы

$title  = 'Сайт нашей школы';
$header = "$welcome, Гость!";
$id = strtolower(strip_tags(trim($_GET['id')));

switch($id) {
   case 'about':
      $title = 'О сайте';
      $header = 'О нашем сайте';
   break;
   case 'contact':
      $title = 'Контакты';
      $header = 'Обратная связь';
   break;
   case 'table':
       $title = 'Таблица умножения';
       $header = 'Таблица умножения';
   break;
   case 'calc':
       $title = 'Он-лайн калькулятор';
       $header = 'Калькулятор';
    break;
}

3. Между тэгамии <title></title> напишите:
<?php echo $title ?>

4. В блоке <!-- Заговолок --> между тэгами <h1></h1> Напишите:
<?php echo $header ?>

5. В боке <!-- Область основного контента --> удалите всё содержимое и
   напишиите:
   <?php
   switch($id) {
        case 'about':
            include 'about.php';
         break;
         case 'contact':
           include 'contact.php';
         break;
         case 'table':
             include 'table.php';
         break;
         case 'calc':
             include 'calc.php';
         break;
         default:
            include 'index.inc.php';
   }
?>

6. Сохраните файл index.php
7. Посмотрите результат в браузере. Обратите внимание, что файл table.php
   Пока не готов к использованию


Упражнения 4: Допиливаем таблицу умножения

1. В текстовом редакторе откройте файл "table.php"
2. Удалите всё, кроме <!--Область основного контента -->
3. В верхней части файла напишите:
<?php

if($_SERVER['REQUEST_METHOD'] == 'POST') {
   $cols = abs((int) $_POST['cols']);
   $rows = abs((int) $_POST['rows']);
   $color = trim(strip_tags($_POST['color']));
}

$cols  = ($cols) ? $cols : 10;
$rows  = ($rows) ? $rows : 10;
$color = ($color) ? $color : 10;

?>

4. В блоке <!-- Таблица --> исправьте вызов функции на:

drawTable($cols, $rows, $color);

5. В значении атрибута "action" тэга <form> напишите:
action="?id=table" method="POST"
