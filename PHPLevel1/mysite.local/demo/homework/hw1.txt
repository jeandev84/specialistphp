# Вывод значения директивы PHP post_max_size
N.B!!! "post_max_size" отвичает за ограничение максимум размер данных отправленно методом POST

1. Вывести значение директивы php.ini "post_max_size" в байтах
2. Откройте в тексте редакторе файл contact.php
3. В самом начале блока <!-- Область основного контента --> напиишите:
<?php
// Вывести значение директивы php.ini "post_max_size" в байтах
// Получаем значение "post_max_size" из PHP.INI
// ini_get("post_max_size") возвращает значение в мега байт на ТЕКУЩЕМ ЛОКАЛНЕ МАШИНЕ (например 250M)

$config_php_ini = ini_get_all(); получает все значения из PHP.INI
$size = ini_get("post_max_size"); // string
$size = (int)ini_get("post_max_size"); // int
?>
4. Задайте два вопроса, необходимые для решения задачи
5. Создайте переменную $size, которая будет содержать текущие значение
директивы post_max_size

6. Получите данные, о величине в которой представленною значение (т.е.
килобайты, мегабайты ...)

7. Используя управляющую конструкцию "switch" вычислите результат и
сохраните его в переменную $size

8. После закрывающего тэга веб-формы </form> напишитеЖ
<p>Максимальный размер отправляемых данных <?= $size ?></p>

9. Сохраните файл "contact.php"

10. Посмотрите результат в браузере
